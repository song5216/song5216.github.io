# 欢迎来到我的博客

你好！欢迎来到我的个人博客网站。

这是一个基于 [Hugo](https://gohugo.io/) 和 [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 主题搭建的博客网站，部署在 GitHub Pages 上。

## 功能特性

这个博客网站包含以下功能：

- ✅ **中文支持** - 完全支持中文内容
- ✅ **分类和标签** - 方便组织和管理文章
- ✅ **搜索功能** - 快速找到你想要的内容
- ✅ **评论系统** - 使用 Giscus 进行评论互动
- ✅ **响应式设计** - 适配各种设备
- ✅ **暗黑模式** - 支持自动切换主题

## 开始写作

使用以下命令创建新文章：

```bash
hugo new posts/文章标题.md
```

然后编辑文章内容，记得将 `draft: false` 设置为 `false` 来发布文章。

## 部署

网站通过 GitHub Actions 自动部署，当你推送代码到 main 分支时，会自动构建并部署到 GitHub Pages。

祝你写作愉快！
